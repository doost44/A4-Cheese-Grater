# GrateTogether — A Value Sensitive Redesign of the Cheese Grater

A portfolio case study for GBDA 306 (Value Sensitive Design), documenting the semester-long redesign of a cheese grater into a dual-mode countertop workstation that serves both children (safety, independence) and cooking enthusiasts (mastery, hedonism).
